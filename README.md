## Requisitos:
- Symfony 2.3 ou superior
- [KnpPaginatorBundle](https://github.com/KnpLabs/KnpPaginatorBundle)

## Instalação e configuração:

Você poderá adicionar facilmente em seu arquivo composer.json

```json
{
    "require": {
        "mero/base-bundle": "dev-master"
    }
}
```
